# Kasparro Agentic Content System

A modular multi-agent pipeline that converts raw product data into structured JSON content pages.

## Features
- Independent agents for analysis, FAQ generation, product page creation, and comparison
- Custom templates with strict JSON rules
- Reusable content logic blocks
- Pydantic-validated structured output

## Outputs
- `product_page.json`
- `faq.json`
- `comparison_page.json`

## Highlights
- Zero-hallucination prompts
- Modular LCEL orchestration
- Fully machine-readable JSON output
